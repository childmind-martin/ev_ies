# Park IES TD3/PPO Experiments

This folder contains the uploaded source code and the data files needed to run the TD3/PPO park integrated energy system experiments.

Included:

- Python source files for data generation, environment logic, training, testing, and plotting.
- `yearly_data_sci.csv`
- `ev_data_s123_bilevel.npy`
- `yearly_data_sci.npy`
- `yearly_data_sci_set_mask.npy`

Excluded:

- Trained models under `models/`
- Runtime logs under `logs/`
- TensorBoard runs under `tb/`
- Generated result exports under `results/`
- Python caches under `__pycache__/`
- Generated image outputs
